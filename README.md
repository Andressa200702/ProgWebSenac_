# AWX GYM - Site Institucional

Este projeto foi desenvolvido como atividade acadêmica para praticar:

- Estruturação HTML semântico
- Organização com section, header, main e footer
- Incorporação de vídeo
- Estrutura de navegação

## Tecnologias utilizadas

- HTML5
- CSS3 (em desenvolvimento)

## Objetivo

Criar um site institucional para uma academia fictícia chamada AWX GYM.
